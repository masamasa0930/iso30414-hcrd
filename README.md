# ISO 30414 人的資本レポート作成システム

ISO 30414:2025（人的資本のレポート及び開示）附属書Aの算出式（A.1〜A.57）に基づき、
素材数値を入力すると指標の自動計算・グラフ化・開示レポート生成を行う単一HTMLツール。

## 使い方

- **公開URL（関係者限定・GitHub Pages）**: https://masamasa0930.github.io/iso30414-hcrd/
  - アクセスにはパスワードが必要（関係者に別途共有。「このブラウザで記憶する」で次回から自動ログイン）
- ローカル: `app.html` をブラウザで開くだけ（サーバー不要・完全ローカル動作・パスワード不要）
- ヘッダーの「デモデータ」で従業員約128名の中小企業3年分のサンプルを読込可能

リポジトリ: https://github.com/masamasa0930/iso30414-hcrd （mainへのpushで自動反映・約1分）

## アクセス制限のしくみ（関係者限定）

- 公開される `index.html` は、アプリ本体（`app.html`・非公開）を **AES-256-GCM で丸ごと暗号化**したもの。
  パスワードから PBKDF2（SHA-256・31万回）で鍵を導出してブラウザ内で復号するため、
  ページのソースを見ても中身は読めない（単なるJSの目隠しではない）。
- `app.html`（平文マスター）は `.gitignore` で除外されておりリポジトリには存在しない。このMacにのみある。
- パスワード変更・アプリ更新時は次を実行して push する：
  ```
  node build.mjs "新しいパスワード"
  git add index.html && git commit -m "update" && git push
  ```
  ※パスワードを変えると全員の「記憶」も無効になる。パスワード自体はリポジトリに書かないこと。
- 入力データは従来どおり各利用者のブラウザ内（localStorage）にのみ保存され、サーバーには送信されない。noindex設定済み。

## 機能

| タブ | 内容 |
|---|---|
| ダッシュボード | JOBScope HRキャピタルダッシュボード風のカードUI。「登録済み指標 N/N」「目標達成/未達」サマリー、11領域のカテゴリ絞り込みチップ、カードごとに大数値＋前年度比矢印＋グラフ（年度別棒/エリア/ドーナツ〔年齢・性別構成〕/リングゲージ〔スコア系〕） |
| データ入力 | 11のHCA領域別に素材数値を年度別（3年分）入力。附属書Aの変数名を併記 |
| 指標一覧 | 全指標の参照番号・算出式・3年推移・目標値入力（SME/大企業モードで表示切替） |
| 開示レポート | 企業公開型「Human Capital Report / People Fact Book」形式：表紙→目次→①トップメッセージ→②人的資本経営の考え方（附属書E準拠TCFD型：ガバナンス/戦略/リスクと機会）→③ハイライト→④領域別の開示（**データのある全指標をグラフ化したカラフルなチャートグリッド**＋指標表＋ギャップ分析＋定性説明）→⑤経年データ集→⑥ISO 30414対応状況→総括。印刷/PDF出力で配布用レポートに |

配色: 11のHCA領域ごとにカラーコード（労働力=青、ダイバーシティ=ピンク、コスト=アンバー、生産性=エメラルド等）を割当。
ダッシュボードのカード（上枠線・カテゴリチップ・グラフ）は領域色、レポートのチャートグリッドはビビッドパレットを循環使用。
棒グラフは年度が新しいほど濃く表示。

デザイン参照: グラフ表現は [JOBScope HRキャピタルダッシュボード](https://jobscope.ai/hrcapital/)、
レポート構成は [コトラ 人的資本レポート一覧](https://consulting.kotora.jp/human-capital/hcr/) に掲載の
各社Human Capital Report（People Fact Book）の形式を参考にしている。

## データの扱い

- 入力データはブラウザのlocalStorageに自動保存（外部送信なし）
- 「保存(JSON)」で顧客別のデータファイルをエクスポート、「読込」で復元
- 顧客の実データを扱う場合はJSONファイルの保管場所に注意

## 実装済みの算出式

附属書Aの式 (A.1)〜(A.57) のうち、単一数値で表現可能な約51指標を実装。
苦情・人権問題・懲戒・係争（A.7.7〜A.7.10）は件数指標として実装。
TRIR (A.16) など代替式の一部と、表形式のみの指標（表A.6等の内訳）は簡略化している。

出典: ISO 30414:2025（JSA英和対訳版）。指標の解釈は規格原文に準拠のこと。
